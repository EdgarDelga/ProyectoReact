    "Learn React Once and Write Everywhere"

    ¿Qué es?

    Se trata de una librería web para interfaces de usuario. Como tambien el uso individual de un nuevo concepto el cual se le conoce como "Componentes", esto dentro de Javascript.

    ¿Para qué sirve REACT?

    React, principalmente se usa para desarrollar páginas web de una manera gratuita y sencilla, gracias a los llamados componentes reutilizables. Estos hacen que sea posible utilizar un mismo elemento en diferentes partes o si es requerido en otros sitios web, sin la necesidad de que se vuelva a escribir su código, haciendo que la cantidad de trabajo y tiempo sea reducida notoriamente

    Componentes

    Se trata de un elemento nuevo dentro de REACT, el cual se utiliza como una pieza reutilizable estos pueden contener tanto modelo BackEnd (Recursos Lógicos), como también unna interfaz de usuario, haciendo muy versatil esto.

    Ejemplos de un Componente

    // Esto se le conoce como una Función, se encuentra en Javascript

    function App() {

    return (

    <div className = "App" >

    <p> ¡Hola mundo en REACT! </p>

    </div>

    ); }

    export default App;

    Esto es un componente dentro de REACT, pero este aún no se ha renderizado dentro de la pagina web

    Renderizado en REACT

    Para poder Renderizar los componentes deseados de REACT, es necesario una función Main la cual llamará una funcion de renderizado, donde se llamarán los elementos deseados

    // Las importaciones son muy importantes para React, son librearías de importante uso

    import React from 'react';

    import ReactDOM from 'react-dom';

    import App from '/assets/App.jsx';

    const Main = () => {

    <App/>

    }

    Esta función (La cual se encontrará en otra pagina), será la que llame de forma principal al componente previamente construido, dando como resultado un "¡Hola Mundo en REACT!"
