    ¿Qué es?

    Se trata de una librería web para interfaces de usuario. Como tambien el uso individual de un nuevo concepto el cual se le conoce como "Componentes", esto dentro de Javascript.

    ¿Para qué sirve REACT?

    React, principalmente se usa para desarrollar páginas web de una manera gratuita y sencilla, gracias a los llamados componentes reutilizables. Estos hacen que sea posible utilizar un mismo elemento en diferentes partes o si es requerido en otros sitios web, sin la necesidad de que se vuelva a escribir su código, haciendo que la cantidad de trabajo y tiempo sea reducida notoriamente

    Componentes

    Se trata de un elemento nuevo dentro de REACT, el cual se utiliza como una pieza reutilizable estos pueden contener tanto modelo BackEnd (Recursos Lógicos), como también unna interfaz de usuario, haciendo muy versatil esto.

    Ejemplos de un Componente

    // Esto se le conoce como una Función, se encuentra en Javascript

    function App() {

    return (

    <div className = "App" >

    <p> ¡Hola mundo en REACT! </p>

    </div>

    ); }

    export default App;

    Esto es un componente dentro de REACT, pero este aún no se ha renderizado dentro de la pagina web

    import React from 'react';

    import ReactDOM from 'react-dom';

    import App from '/assets/App.jsx';

    const Main = () => {

    }

    Historia de REACT

    Jordan Walke siendo un ingeniero de Software en Facebook, liberó el primnero prototipo de React llamado "FaxJS", altamente influenciado por XHP de HTML, este fue utilizado por primera vez en el feed de noticias en 2012.

    • 2010: Facebook introduce XHP a su stack PHP como código abierto
    • 2013: Facebook lanza REACT como código abierto
    • 2013: React y JSX se encuentra disponibles para aplicaciones de Python
    • 2015: Netflix anuncia que usará REACT para su desarrollo de interfaz de usuario
    • 2017: Existen diversos programas que reportan errores en REACT para su constante mejora

    ¿Como usar?

    Cuando se empieza un proyecto con React, una simple página HTML con una etiqueta de tipo script puede ser la mejor opción. Solo toma un minuto para configurarse. A medida que tu aplicación crezca, puede que consideres una configuración más integrada. Hay una gran variedad de herramientas que recomendamos para aplicaciones más grandes. Cada una de ellas puede funcionar con poca configuración, y te permite tomar toda la ventaja de un ambiente React completo.

    React te ayuda a crear interfaces de usuario interactivas de forma sencilla. Diseña vistas simples para cada estado en tu aplicación, y React se encargará de actualizar y renderizar de manera eficiente los componentes correctos cuando los datos cambien.

    Un componente simple

    Los componentes de React implementan un método llamado render() que recibe datos de entrada y retorna qué mostrar.

    Un componente con estado

    Además de obtener datos de entrada (a los que accedes a través de this.props), un componente puede tener datos en su estado interno (a los que accedes a través de this.state). Cuando los datos del estado de un componente cambian, se vuelve a invocar render con los nuevos valores en this.state.

    Una aplicación

    Usando props y state, podemos integrar todo en una pequeña aplicación de tareas pendientes. Este ejemplo usa state para tener un control de la lista actual de objetos así como el texto que el usuario ha ingresado. A pesar de que los manejadores de eventos parecen ser renderizados en línea, serán recolectados e implementados usando delegación de eventos.
